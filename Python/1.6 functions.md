# âœ… 4) Function with Return Value âœ…

If you want your function to **send back a result**, use `return`.

```python
def add(a, b):
    return a + b
```

### âœ… Example

```python

print(result)
```

âœ… Output:

```
30
```

âœ… Senior Rule:

* Use `print()` for showing output to user
* Use `return` for actual program logic
